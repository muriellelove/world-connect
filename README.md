# 🌍 WorldConnect — Angular/Ionic App

WorldConnect est une application complète construite avec Angular et Ionic. Elle propose une interface utilisateur moderne pour explorer les pays du monde, obtenir la météo, traduire du texte, et gérer son profil via une authentification simple.

## Fonctionnalités :
- Authentification utilisateur
- Recherche de pays
- Météo en temps réel
- Traduction de texte
- Tableau de bord personnalisé

## Objectifs :
Apprendre les meilleures pratiques Angular/Ionic, consommer des API REST, et bâtir une app modulaire et responsive.

## APIs utilisées :
- ReqRes
- REST Countries
- OpenWeatherMap
- LibreTranslate
